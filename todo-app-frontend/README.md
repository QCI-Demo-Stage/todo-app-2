# Todo App — Frontend

React + TypeScript client for **Todo App 2**, built with [Vite](https://vite.dev/). It will provide the responsive UI for creating, listing, editing, and deleting tasks, and will call the project’s REST API.

## Scripts

| Command        | Description                    |
| -------------- | ------------------------------ |
| `npm run dev`  | Start the dev server (HMR)     |
| `npm run build`| Production build               |
| `npm run lint` | ESLint (Airbnb + TypeScript)   |
| `npm run format` | Prettier on `src`          |
| `npm run preview` | Preview production build   |

## Structure

- `src/components` — UI components
- `src/hooks` — custom React hooks
- `src/store` — app state
- `src/styles` — shared styles

Backend API and repo-wide docs live at the repository root.
